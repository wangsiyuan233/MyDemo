### 任务目的
- 生成一个新的右键事件

### 任务描述
- oncontextmenu()事件在元素中用户右击鼠标时触发并打开上下文菜单。

### 任务注意事项
- oncontextmenu()事件结束后 要关闭默认的右键事件；
- 鼠标点击的位置，就是新菜单的起点位置。
